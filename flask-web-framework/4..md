# 4. 라우팅

**라우팅(Routing)**

어떤 주소를 누가 담당할 것인가, 어떤 요청을 어떤 함수가 응답할 것인가를 연결 시켜주는 작업

![](../.gitbook/assets/KakaoTalk\_20220509\_134939692.jpg)

URL 경로 상에서 변하는 부분이 있다면 <변수이름:데이터타입>의 형식으로 패턴을 지정

```
from flask import Flask
import random

app = Flask(__name__)

@app.route('/')
def index():
    return 'Welcome'

@app.route('/create')
def create():
    return 'Create'

@app.route('/read/<id>')
def read(id):
    return 'Read ' + id

app.run(port=8080, debug=True)
```
